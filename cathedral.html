✦ Deployment Protocol

  1. Clone each repository and add the common files from the blueprint.
  2. Place master images in assets/raw/ (PNG/JPG only).
  3. Run: npm install then npm run build.
  4. Deploy to Netlify: netlify deploy --prod --build.
  5. In Netlify UI: Disable "Image Asset Optimization".
  6. Update tesseract-bridge/registry/realm_links.json with live URLs.
  7. Verify /core/health-check.html returns 200 on all sites.
  8. Add nodes/combinations to registry/nodes.json (schema enforced).
  9. For content updates: Bump SW_VERSION in public/sw.js.
